## אפליקציית מובייל למעקב תרגול תלמידים

- נוסף מסלול אפליקציית PWA חינמית למובייל על בסיס GitHub Pages.
- מיקום מומלץ בריפו: `mobile-pwa/`
- מטרת האפליקציה: שליפה חיה של קובצי Excel/CSV מהריפו הציבורי, חישוב זמן תרגול יומי נטו ללא הפסקות, והצגת נתונים לפי תלמיד/כיתה.
- מסכים באפליקציה: `יומי`, `לפי תלמיד`, `לפי כיתה`, `כללים`, `הגדרות`.
- כל הכפתורים, הניווט והטפסים בעברית RTL.
- כל רשומה יומית תכלול:
  - שם תלמיד
  - כיתה/קבוצה
  - תאריך
  - משעה / עד שעה
  - זמן נטו ללא הפסקות
  - רשימת משימות שבוצעו באותה יממה
  - מקור הנתון: קובץ / גיליון / שורה
- כלל חישוב מחייב:
  - הפסקות אינן נחשבות זמן תרגול.
  - יש להציג שורה אחת לכל תלמיד לכל יממה.
  - אין להציג את שם המרחב, אלא רק את שם המשימות.
- מקור אמת לנתונים:
  - קובצי Excel/CSV בריפו עצמו בלבד.
- פרסום:
  - האפליקציה מיועדת לפרסום דרך GitHub Pages בלבד, ללא שירות בתשלום.
